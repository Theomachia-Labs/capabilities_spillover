What is this calculating?

Goal: Determine which of the 4 uncertainty question formats produces the most informative uncertainty values — i.e., which type's stated uncertainty best predicts how much a model's scores actually vary.

How it works:

  1. Normalize uncertainties — All 4 types are converted to a 0–1 scale. Types 1 & 2 are already 0–1. Type 3 (range [low, high]) becomes (high − low) / 5. Type 4 (evidence string) is mapped to fixed values ("Strong direct" → 0.1, "Some indirect" → 0.4, etc.).
  2. Compute actual score variance — For each (model, paper, dimension), we measure how much the model's scores vary across repeated attempts. "Combined" uses all 12 scores (3 attempts × 4 types); "Per-type" uses only the 3 scores within each type.
  3. Correlate — For each uncertainty type, we compute the Pearson correlation between stated uncertainty (X) and actual score variance (Y) across all data points. A positive correlation means the model "knows when it doesn't know" — high stated uncertainty corresponds to inconsistent scores.

Interpreting results: The type with the highest positive Pearson r is best-calibrated. r ≈ 0 means the uncertainty is noise. r < 0 means the model is overconfident precisely when it should be uncertain.

Uncertainty Type Definitions
  • Type 1: Uncertainty (0-1) — A 0-1 rating where 0 means complete certainty and 1 means a complete guess.
  • Type 2: Probability off by 2+ — The probability that the "true score" could be 2 or more points away from the guess.
  • Type 3: Plausible range — The lowest and highest scores that are considered plausible (e.g., [2, 4]). Larger ranges indicate higher uncertainty.
  • Type 4: Evidence level — How much evidence the score was based on: Strong direct evidence, Some indirect evidence, Very little evidence, or Pure guess.
